# Institute of Digital Risk (IDR) - Website Assignment

A modern, responsive single-page website for the Institute of Digital Risk, showcasing their mission, services, and training programs in digital and AI risk management.

## 🌐 Live Demo

[View Live Website](your-deployment-link-here)

## 📋 Project Overview

This project was developed as an assignment for a well-known company, demonstrating proficiency in modern web development using vanilla HTML, CSS, and JavaScript. The website serves as the digital presence for IDR, an industry-led training and deployment institute focused on digital, cyber, and AI risk.

## ✨ Features

### Design Elements
- **Custom Logo Design**: Two variants (icon-only and full logo with text)
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, minimalist design with orange, white, and black color palette
- **Smooth Animations**: Hover effects, transitions, and scroll behaviors

### Website Sections

1. **Hero Section**
   - Compelling headline reflecting IDR's mission
   - Clear value proposition
   - Dual call-to-action buttons

2. **About IDR**
   - Detailed explanation of IDR's role as an industry-led institute
   - Emphasis on academic-industry partnership
   - Visual stat card highlighting collaboration

3. **Service Pillars**
   - Three core service areas: Academy, Innovation, Advisory Services
   - Interactive cards with hover effects
   - Visual pipeline showing "Train → Hire → Innovate → Deploy" model

4. **Community**
   - Target audience overview
   - Focus on financial services and critical infrastructure sectors

5. **Contact Section**
   - Functional interest registration form
   - Client-side validation
   - Email contact alternative

## 🚀 Getting Started

### Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/idr-website.git
   cd idr-website/Assignment
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js http-server
     npx http-server
     
     # Using VS Code Live Server extension
     Right-click index.html → "Open with Live Server"
     ```

3. **View the site**
   - Navigate to `http://localhost:8000` (or your server's port)
